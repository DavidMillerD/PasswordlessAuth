# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2025-03-02

### Added
- Express.js middleware with plug-and-play authentication
- Comprehensive session management system
- Advanced rate limiting with per-endpoint configuration  
- Token blacklist functionality for secure logout
- Enterprise-grade logging system with security event tracking
- Custom error handling with structured error codes
- Admin statistics and monitoring endpoints
- Complete Express.js server example

### Enhanced
- Updated documentation with advanced usage examples
- Added TypeScript declarations for Express integration
- Improved error messages and debugging information
- Enhanced security features and best practices guide

### Security
- Multi-layer rate limiting protection
- Session-based security with device tracking
- Token revocation and blacklisting
- Comprehensive audit logging
- Security event monitoring and alerting

## [0.1.0] - 2025-02-19

### Added
- Initial release of PasswordlessAuth library
- Magic link authentication via email
- WebAuthn biometric authentication support
- JWT token management and validation
- TypeScript support with full type definitions
- Basic usage examples
- Comprehensive documentation

### Security
- Secure token generation using crypto.randomBytes
- JWT signature verification
- Email validation
- WebAuthn origin validation
